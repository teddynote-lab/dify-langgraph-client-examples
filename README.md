# Quick-start LLM App Client (for Dify & LangGraph)

Dify 와 LangGraph 를 배포하기 위한 프론트엔드 유스케이스를 소개하는 레포지토리입니다.

프론트엔드에 대해 잘 모르는 개발자도 사용하기 쉽게 가이드를 제공합니다.

### 2. 웹사이트 내부에 삽입 - Embedding In Websites
